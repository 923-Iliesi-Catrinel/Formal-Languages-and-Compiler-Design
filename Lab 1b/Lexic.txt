Cat Programming Language Specifiion

Alphabet:

a. Upper case (A-Z) and lower case (a-z) letters of the English alphabet
b. Special characters: @ _ , : ~
c. Decimal digits (0-9)

Lexic:

a. Special symbols, representing:
    - operators: add sub mul div mod < <= < >= = -> <--> not and or
    - separators: [ ] { } space
    - comments ~
    - reserved words:
        - function ::= fn
        - assignment ::= ->
        - input/output ::= read/write
        - condition ::= if/otherwise
        - loop ::= loop until/loopfor/loopwhile/break
        - types ::= integer/boolean/character/const/void/array/string
        - return ::= ret

b. identifiers
    - a sequence of letters and  digits, such that the first character is @, followed by at least a letter; the rule is:
        identifier ::= "@"letter | "@"letter{letter}{digit}
        letter ::= "A" | "B" | ... | "Z" | "a" | "b" | ... | "z"
        digit ::= "0" | "1" | ... | "9"

c. constants
    1. integer - rule:
        noconst ::= "+"no | "-"no | no
        no ::= digit{no}

    2. character
        character ::= 'letter' | 'digit'

    3. string
        constchar ::= "string"
        string ::= char{string}
        char ::= letter | digit
